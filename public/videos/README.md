## 🎥 视频要求

为了获得最佳播放效果，建议视频满足以下要求：

- **格式**: MP4 (H.264编码)
- **分辨率**: 1920x1080 或 1280x720
- **比例**: 16:9
- **文件大小**: 建议 < 50MB (确保快速加载)
- **时长**: 建议 < 3分钟

## 🚀 优化建议

1. **视频压缩**: 使用 HandBrake 或在线工具压缩视频
2. **格式转换**: 确保为 MP4 格式以获得最佳兼容性
3. **画质平衡**: 在文件大小和画质之间找到平衡点
